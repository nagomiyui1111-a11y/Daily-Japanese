# Daily Japanese for Beginners
A beginner-friendly Japanese learning website with daily vocabulary, conversation sentences, kanji practice, mini quizzes, and downloadable PDF worksheets.
## Features
- **Daily Vocabulary** – 30 days of vocabulary with practice.
- **Conversation Sentences** – 30 days of daily conversation examples.
- **Kanji Practice** – 30 days of kanji exercises.
- **Mini Quizzes** – Fun quizzes to test your knowledge.
- **PDF Worksheets** – 4-page worksheet for offline practice.
- **WhatsApp Channel Info** – How to join and learn daily.
## How to Use
1. Open `index.html` in your browser.
2. Use the navigation buttons to explore sections.
3. Download the PDF worksheets for practice.
4. Visit the WhatsApp channel for daily Japanese lessons.
